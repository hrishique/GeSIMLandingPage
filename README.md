# GeSIM Landing Page

Welcome to the official landing page repository for **GeSIM**!

## What is GeSIM?

**GeSIM** is a revolutionary platform designed for digital nomads, travelers, and global citizens. GeSIM provides seamless, global eSIM connectivity, allowing you to stay connected anywhere in the world without the hassle of physical SIM cards or expensive roaming charges. Join a thriving global community and experience the freedom of borderless mobile connectivity.

## About This Repository

This repository contains the source code for the GeSIM landing page, built with [Next.js](https://nextjs.org/). The landing page introduces GeSIM, showcases its features, and allows users to join the waitlist for early access.

### Features

- Modern, responsive design
- Waitlist signup form
- Interactive UI components
- Built with Next.js, React, and Tailwind CSS

## Getting Started

To run the landing page locally:

1. **Clone the repository:**
   ```
   git clone https://github.com/your-org/gesim-landing.git
   cd gesim-landing
   ```

2. **Install dependencies:**
   ```
   npm install --legacy-peer-deps
   ```

3. **Run the development server:**
   ```
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the landing page.
